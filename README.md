# Image and Video Processing Assignments

This repository contains coursework for the **Image and Video Processing** course.

It includes the notebook, code, written explanations, and visual outputs used to solve the assignment tasks. The repository is organized so the notebook can be run end-to-end in a standard Python/Jupyter environment.

## Repository Structure

```text
image-and-video-processing-master/
├── README.md
└── assignment_1/
    └── assignment_1_completed_cleaned.ipynb
```

## Assignment Topics

The current notebook covers the following image processing tasks:

- **Tone mapping and linearization**
- **Color correction / white balancing**
- **Histogram analysis**
- **Histogram equalization**
- **Image matting**

## Requirements

The notebook uses common Python libraries for scientific computing and image processing:

- Python 3.10+
- Jupyter Notebook or JupyterLab
- NumPy
- Pillow
- Matplotlib

Install the dependencies with:

```bash
pip install numpy pillow matplotlib notebook
```

## How to Run

1. Clone the repository:

```bash
git clone <your-repository-url>
cd image-and-video-processing-master
```

2. Start Jupyter:

```bash
jupyter notebook
```

3. Open the notebook:

```text
assignment_1/assignment_1_completed_cleaned.ipynb
```

4. Run the cells from top to bottom.

## Notes

- The notebook is structured to be readable and reproducible.
- Markdown cells explain the goal of each section and summarize the results.
- Code cells include comments to make the workflow easier to follow.
- Make sure all required input images are available in the expected paths before running the notebook.

## Purpose

This repository is intended for academic use and assignment submission. It documents the implementation and results for image processing tasks in notebook form.
